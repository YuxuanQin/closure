## 构建说明 (命令行界面)

（注意，这是针对命令行下的编译指导，如果用的是 IDE，可以略过）

要是下了完整的 TeXLive，构建应该不会有什么大问题，不过要注意，由于种种因素，存在目录的 `.tex` 文件需要**多次编译**！

在终端中执行至少两次

```
xelatex paper.tex
```

运行时，可能会出来以问号开头的问题，那种一路回车就可以了。

针对代码本身的说明请看注释。
